# The Zero-Inference Kernel 🦞

A kernel that performs zero reasoning itself, acting only as a cryptographic switch for delegating reasoning to verified modules. 

- **Authority:** It owns the I/O and the Keys.
- **Sterility:** It never performs inference.
- **Verification:** It acts as the ultimate reference monitor, separating the 'Thought' from the 'Mechanism'.
